# Ames-House-Prices-Prediction
Predicting the final price of a house based on its features and characteristics, using Machine Learning techniques
![Background image](docs/background.png)

## Project Description
This project focuses on predicting the price of a house in Ames,Iowa. From the given dataset, we are required to predict the value of the SalePrice variable based on factors such as Street, and SaleCondition using Machine Learning techniques.
